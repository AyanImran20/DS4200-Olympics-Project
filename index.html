Visual Analysis of Olympic Games History and Athlete Performance

An interactive exploration of Summer Olympic medal history, country performance, and gender participation trends from 1896 to the present.

Project Goal

Analyze the history of the Summer Olympic Games using data visualization to reveal patterns in medal counts, athlete demographics, and country-level performance across time.

Key Questions

  • How have medal counts evolved for top countries over time?
  • How has gender participation changed across Olympic years?
  • Which sports and events dominate medal achievements?

Team Members & Roles

  • Harshini – Data preprocessing & exploratory analysis
  • Mihir – Static visualizations (charts, maps, heatmaps)
  • Ayan – Interactive dashboard & website integration

Data Description

Our primary dataset is summer.csv from the visualization-of-olympic-dataset repository. Each row represents a medal-winning athlete in the Summer Olympics, with attributes such as year, city, sport, event, country (NOC), athlete name, and medal type.

We integrate this file with Dictionary.csv (country metadata, GDP per capita, population) and ISO_Codes.csv (standardized country codes) to create a combined OLYMPICS Dataset that connects athlete performance to country-level characteristics.

Planned Data Properties

  • > 35,000 medal records (observations)
  • > 10 features after merging and feature engineering
  • Mix of categorical variables (sport, event, country, medal)
  • Continuous variables (year, GDP per capita, population)
